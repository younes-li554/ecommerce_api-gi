# E-commerce API

## Setup
1. Clone repo
2. Run `npm install`
3. Create `.env` with your MongoDB URI and JWT secret
4. Run `npm run dev` to start server

## Features
- JWT authentication
- Role-based access control (Admin/User)
- Input validation
- Error handling
- Logging via Winston
- Security hardening with Helmet and rate limiting

See `docs/api.md` for all endpoints and usage.
